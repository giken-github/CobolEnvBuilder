# COBOL実行環境構築スクリプト

COBOL言語をコンパイル、実行ファイルを作成するための環境を構築します。

## 対応環境

- WSL2
- Ubuntu 22.04、24.04

## 環境構築実行

``` sh
git clone https://github.com/giken-github/CobolEnvBuilder.git
cd CobolEnvBuilder/builder
sudo bash build.sh
```
